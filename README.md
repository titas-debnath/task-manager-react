# 📚 React Task Manager App

A simple and clean Task Manager built using React.js.  
This project helps users manage daily tasks efficiently with features like add, delete, complete, search, and persistent storage.

---

## 🚀 Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 🔍 Search tasks  
- 📊 Task counter (Total / Completed / Pending)  
- 💾 Local storage (tasks saved after refresh)  
- 🧩 Component-based architecture  

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3

---

## 📂 Project Structure

- TaskForm (Input + logic)
- TaskList (Displays tasks)
- TaskItem (Single task UI)

---

## 💡 What I Learned

- Props and component structure
- State management
- Array methods (map, filter)
